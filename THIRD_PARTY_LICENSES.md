## Open Source Software

Kestrel DB is built on top of many excellent open source projects. The complete list of dependencies and their licenses can be found in the `package.json` file and their respective LICENSE files in `node_modules/`.

### Major Dependencies

- **Electron** - Cross-platform desktop framework (MIT License)
- **React** - UI framework (MIT License)
- **TypeScript** - Type-safe JavaScript (Apache-2.0 License)
- **Vite** - Build tool (MIT License)
- **Tailwind CSS** - Utility-first CSS framework (MIT License)
- **Monaco Editor** - Code editor from Visual Studio Code (MIT License)
- **TanStack Table** - Headless table library (MIT License)
- **TanStack Virtual** - Virtualization library (MIT License)
- **Zustand** - State management (MIT License)
- **mysql2** - MySQL client for Node.js (MIT License)
- **keytar** - Native credential storage (MIT License)
- **electron-updater** - Auto-update functionality (MIT License)
- **Lucide React** - Icon library (ISC License)

### Full Dependency List

For a complete list of all dependencies and their licenses, you can generate a report using:

```bash
npm list --all
npx license-checker --summary
```
